# Use by CDN

``` html
<script src="https://unpkg.com/cursor-traces@0.0.4/min.js"></script>
```

- put this in your html header part
- in your html body , use the the function `startCursorTraces()`

```html
<script>
...
startCursorTraces();
</script>
```
